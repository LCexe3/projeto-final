# CastraPrev

O **CastraPrev** é um sistema web desenvolvido com foco na castração e prevenção de animais domésticos, como cães e gatos. A proposta do projeto é facilitar a comunicação entre tutores e veterinários, permitindo que o tutor
cadastre seus dados e os dados do animal, solicite agendamentos e acompanhe informações relacionadas ao cuidado e prevenção.

O sistema também possui uma área voltada para veterinários, onde é possível visualizar os cadastros dos tutores e seus respectivos animais, auxiliando no controle, organização e acompanhamento dos atendimentos. 
Além disso, o projeto conta com páginas informativas sobre castração, prevenção, cuidados com os pets e políticas públicas relacionadas à proteção animal.

O objetivo principal do CastraPrev é oferecer uma plataforma simples, acessível e organizada para apoiar tutores e veterinários no processo de cuidado animal, incentivando a posse responsável e a prevenção de 
problemas relacionados ao abandono e à reprodução descontrolada.
